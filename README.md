# Invest India – Stock Market UI

A modern stock market-themed React web app, built using:

- ⚛️ React.js
- 💨 Tailwind CSS
- 🌐 React Router

## Features

- Responsive layout
- Multi-page navigation (Home, Login, About)
- Clean and fast UI

## How to Run

1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
